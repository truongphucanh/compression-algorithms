Multimedia Compression Application

Introduction:

Multimedia Compression Applicaton used for compression and decompress any data. In this application, we only test for text, image with 3 basic algorithms RunLength Coding, Shannon Fano and JPEG Lossless.

The application was built on Web Application. With Front end are HTML, CSS, Semantic UI, jQuery, Angular, Pannellum ... and Back end are Node.js, Python. After receiving the request from Client, Node.js will connect to Python to process the files. After that, It will response the result to Client.
